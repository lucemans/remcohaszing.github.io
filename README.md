# remcohaszing.nl

This repository contains the source code for [remcohaszing.nl](https://remcohaszing.nl). It’s
deployed automatically to GitHub pages.

To start a development server, run:

```sh
npm start
```

To build the website, run:

```sh
npm run build
```
